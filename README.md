# Frogger Game (p5.js)

A recreation of the classic Frogger game built using JavaScript and p5.js.  
This project focuses on animation, collision detection, and game logic.
You objective is to guide your character (the frog), avoid the vehicles and the beaver crossing in the middle, and reach the pond.
You have 3 lives. Die 3 times and the game is over.

## Features
- Player movement
- Obstacles and hazards
- Win/lose conditions
- Object-oriented structure

## Tech
JavaScript, p5.js
